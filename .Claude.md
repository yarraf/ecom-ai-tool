3. Tu obtiens :

   - ✅ Storage (fichiers)

```javascript
// Exemple d'utilisation
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  'https://xxx.supabase.co',
  'ta-clé-anon'
)

// Sauvegarder une description

  .from('descriptions')
  .insert({ 
    user_id: userId,
```

## 💡 Étape 7 : Astuces pour Économiser

### **1. Cache Agressif**
Sauvegarde les résultats pour éviter appels API répétés :
- Cache en mémoire (code fourni)
- Redis gratuit via **Upstash** (10,000 commandes/jour)

### **2. Rate Limiting**
Limite les abus (code fourni) :
- 10 requêtes/heure par IP en gratuit
- 100 requêtes/jour pour users inscrits

### **3. Compression des Prompts**
- Prompts courts = moins de tokens = moins cher
- Utilise des abbréviations claires

### **4. Batch Processing**
Au lieu de 10 appels, fais 1 appel pour 10 produits :
```javascript
// ❌ Mauvais : 10 appels API
products.forEach(p => generateDescription(p))

// ✅ Bon : 1 seul appel
generateBatchDescriptions(products)
```
